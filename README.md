A participatory space to jointly manage assemblies and processes..

## Usage

Superspaces will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-superspaces"
```

And then execute:

```bash
bundle
```
