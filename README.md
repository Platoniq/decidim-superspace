# Decidim::Superspace

A participatory space to jointly manage assemblies and processes..

## Usage

Superspace will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-superspace"
```

And then execute:

```bash
bundle
```
